# Ivy Script

存放一些自己写的脚本文件。

Loon类插件的请自行拼接 `loon://import?plugin=` 打开浏览器输入后导入。

|  类型  |      名称      |                             链接                             |                   说明                    |
| :----: | :------------: | :----------------------------------------------------------: | :---------------------------------------: |
|  Loon  | 酷安链接重定向 | [Coolapk_Redirect.plugin](https://github.com/VenenoSix24/Ivy-Script/raw/refs/heads/main/Loon/plugin/Coolapk_Redirect.plugin) |                                           |
| FFmpeg |    视频压缩    | [video_compress.command](https://github.com/VenenoSix24/Ivy-Script/blob/main/macOS/FFmpeg/video_compress.command) | macOS使用，需安装ffmpeg，参数越大质量越高 |
