## 自签

### 1.相关资源/教程

**[点我查看/README](https://github.com/VenenoSix24/Ivy-Script/blob/main/SideStore/README.md)**

### 2.自签/续签教程

1. 下载 [iLoader](https://iloader.app/) 登录一个不常用的 Apple ID
2. 将手机连接电脑，在 iLoader 中安装 LiveContainer+SideStore（每夜构建版）/ 或者你熟悉的版本
3. 剩下的步骤去B站搜一下有教程，主要讲自动续签（使用 Clash Mi 代替 LocalDevVPN 解决网络问题导致的续签失败）
4. 在 Clash Mi -> 核心设置 -> 覆写 -> 右上角添加 -> 添加配置（[配置链接](https://github.com/VenenoSix24/Ivy-Script/raw/refs/heads/main/SideStore/ClashMi/overwhite.yaml)）
5. 使用自动化定时执行快捷指令实现自动续签（[快捷指令链接](https://www.icloud.com/shortcuts/f176a472a81048f882828bc8e169dc93)）
